CPU Scheduling Algorithms

CPU Scheduling is the process of determining which process in the ready queue should be allocated the CPU next. Various algorithms handle this, each with different strategies and trade-offs. The key metrics to evaluate these algorithms are turnaround time, waiting time, throughput, and CPU utilization.

⏱️

Round Robin

Each process is given a fixed time slice (quantum) in a cyclic order, making it fair and responsive.
⏱️

FCFS

Processes are executed in the order they arrive, without preemption.
⏱️

SJF

Selects the process with the smallest burst time among those that have arrived, non-preemptive.
⏱️

Priority scheduling

Processes are scheduled based on priority; lower number means higher priority
⏱️

SRTN

Preemptive version of SJF that always runs the process with the shortest remaining burst time.
